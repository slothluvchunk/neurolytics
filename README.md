# Neurolytics

A command-line tool for tracking and analyzing neurological health data, starting with seizure logging and pattern detection.

## Project Status

🚧 **Early Development** - This is a learning project. Currently building basic functionality.

## Goals

- Log seizure events (date, time, type, duration, sensation-severity, notes)
- Identify patterns and trends over time
- Generate summaries for medical appointments
- Learn Python development practices (TDD, project structure, data handling)

## Planned Features

- [ ] Add/view seizure records
- [ ] Save/load data from JSON
- [ ] Basic statistics (frequency, timing patterns)
- [ ] Export reports
- [ ] Data visualization
- [ ] Trigger tracking

## Why?

As someone learning a new language, with a long history in and around disability and A11Y spaces, it seemed like a great fit; building a practical tool to learn Python while creating something potentially useful for people managing epilepsy or other neurological conditions.
